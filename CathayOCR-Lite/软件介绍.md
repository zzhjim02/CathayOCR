# CathayOCR Lite (极速版) v1.0

基于 ncnn Vulkan 的便携式 PDF OCR 工具。CathayOCR 专业版的轻量分支。

## 特性亮点

🚀 **单一引擎，一键启动**
- 仅保留 ncnn Vulkan 引擎，无需选择纠结
- GPU 模式（Vulkan 加速）/ CPU 模式（原生 ncnn，不初始化 Vulkan）
- 支持任意品牌显卡：NVIDIA / AMD / Intel 均可
- 支持 PP-OCR 多版本模型（v3 ~ v6）

📦 **极致便携，U 盘即用**
- 体积仅 ~1.4GB，原始版 9GB 的 1/6
- 自带便携 Python 环境，无需任何安装
- 拷贝到任意目录即可运行

🌍 **多语言支持**
- 中文（繁简体自动识别）、English、Français、Deutsch、日本語、한국어
- Español、Italiano、Português、Pусский、Tiếng Việt 等

📄 **PDF OCR 专业功能**
- GPU 双实例并行（提升 30~50%）
- 古籍/竖排/复杂排版支持
- 拖放文件处理，批量输出

## 系统要求

- **操作系统**：Windows 7/10/11 64位
- **GPU 模式**：支持 Vulkan 1.1+ 的显卡
- **CPU 模式**：任意 x64 处理器均可

## 使用方法

1. **解压**到任意文件夹（路径可含中文）
2. 双击 **`启动.bat`**
3. 🎯 简单模式：选择语言 → 点运行
4. 🔧 专业模式：更多参数可调（模型、边长、精度等）
5. 拖放 PDF 文件到列表 → 点运行

## 文件结构

```
D:\一些文件夹\
├── 启动.bat                  # 双击启动
├── _cuda_check.py            # GPU 检测辅助脚本
├── portapython\              # 便携 Python（含 PyQt5、OpenCV 等）
├── ncnn\
│   └── PPOCR-ncnn-Vulkan\    # ncnn 引擎 + 模型库
└── CathayOCR-Lite\
    ├── umi_ocr_pdf_processor_ui.py  # 主程序
    └── 开启 CathayOCR Lite.bat       # 手动启动入口
```

## 故障排除

| 问题 | 原因 | 解决 |
|------|------|------|
| GPU 模式黑屏/崩溃 | 显卡驱动过旧 | 更新显卡驱动 |
| AMD 核显高频崩溃 | FP16 加速不稳定 | 切换到 FP32 精度 |
| 简单模式 GPU 自动选了 CPU | 未检测到独显 | 手动切换为「独显」 |
| 无法启动 | 安全软件拦截 | 加入信任区或命令行运行查看错误 |
